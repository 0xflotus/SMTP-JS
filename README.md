# SMTP-JS

Just an app that sends emails to everyone.

### Installation

Here's how to install the smpt and make it work (in cmd).

- ``cd directory``
- ``npm i``
- config the file `infos.json` & `mails_list.txt` in SMTP-CONFIG
- ``node index.js``
